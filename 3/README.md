# 3  12345 Hello World！

```java
public class HelloWorld//这行改了个名字
{
    public static void main(String []args)//这行照抄
    {
        System.out.println(12345);
        System.out.println("Hello World！");
    }
}
```

使用System.out.println\(\)来打印出括号中的 12345 和 Hello World！

打印结果为：

```text
12345
Hello World！
```

可以自己在在线编译器中尝试一下
