# 7  方法（选修）

先看一个不太简短也不太聪明的程序：

```java
public class Method
{
    public static void main (String []args)
    {
        int i = 1;
        Shuchu(i);
    }
    
    void Shuchu(int a)
    {
        System.out.println(a);
    }
    
}
```

输出结果：

`1`

上面的程序主要是为了讲解方法的用法，于是就把一段简单的代码复杂化了。

在这段程序中，我们定义了一个方法Shuchu\(\)，小括号里是数据类型，大括号里是要执行的代码
